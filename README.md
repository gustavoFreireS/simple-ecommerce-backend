# Simple Ecommerce Backend

## Installation

- `yarn`
- `cp .env.example .env`
- on the .env file change the `DATABASE_URL` configuration
- `yarn generate`
- `yarn dev`
- open http://localhost:5000

## after change the schema.prisma

- `yarn prisma generate`

## using docker

- `docker-compose up -d`

## using pgweb

- http://localhost:8888

## using prisma studio instead

- `yarn prisma studio`
